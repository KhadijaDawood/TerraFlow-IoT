# TerraFlow IoT: Smart Automated Irrigation & Micro-Climate System

An advanced, data-driven precision agriculture and automation solution developed to maximize water conservation and optimize crop yields using embedded systems and IoT architecture.

## 🛠️ Integrated Hardware Framework
* **Arduino UNO R3** (Central Microcontroller Architecture)
* **Capacitive Soil Moisture Sensor v1.2** (Corrosion-resistant analog tracking)
* **5V Single-Channel Relay Module** (Automated high-voltage isolation switch)
* **12V DC Submersible Water Pump** (Fluid mechanics actuator)
* **High-Capacity 9V Power Core** (Autonomous remote power source)

## 🚀 Firmware Automation Logic Summary
* **Dynamic Moisture-Calibration Thresholds:** Built-in dual logic parameters (Dry/Wet bounds) to prevent intermittent relay chattering and protect pump motor lifespan.
* **Low-Power Remote Optimization:** Programmed non-blocking data cycle updates to ensure operational battery efficiency in real-world agricultural fields.
* **Serial Telemetry Diagnostics:** Full hardware stream outputs through the Serial Monitor for instant real-time troubleshooting.
